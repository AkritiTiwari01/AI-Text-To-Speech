# 🎙️ AI Text to Speech Converter

A beautiful, modern Text-to-Speech web app built with HTML, CSS, and JavaScript — featuring voice effects, waveform animation, and a stunning dark UI.

## ✨ Features

- 🔊 Convert any text to speech instantly
- 🎤 Voice effects — Male, Female, Child, Default
- 🎛️ Fine-tune Pitch and Speed with sliders
- 🌊 Animated waveform visualizer while speaking
- 🔁 Stop/replay anytime
- 📱 Fully responsive design

## 🛠️ Built With

- HTML5 — Structure
- CSS3 — Styling & animations
- JavaScript — Web Speech API logic
- Web Speech API — Text to speech engine

## 🚀 How to Run

1. Clone the repository:
```bash
   git clone https://github.com/AkritiTiwari01/ai-text-to-speech.git
```
2. Open the project folder
3. Simply open `index.html` in your browser — no server needed!
